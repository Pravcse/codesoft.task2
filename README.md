# CODSOFT Task 2 – Simple Calculator

## Description

This project is a **Simple Calculator** developed in **C++** that performs the four basic arithmetic operations: **Addition, Subtraction, Multiplication, and Division**. It is a console-based application designed for beginners to practice fundamental C++ programming concepts such as user input, conditional logic, switch-case statements, and arithmetic operators.

## Features

* Perform Addition
* Perform Subtraction
* Perform Multiplication
* Perform Division
* Simple and user-friendly console interface
* Fast and lightweight application
* Beginner-friendly source code

## Technologies Used

* C++
* Standard Input/Output (`iostream`)
* Switch-Case Statements
* Arithmetic Operators

## How It Works

1. Enter the first number.
2. Enter the second number.
3. Select the desired arithmetic operation:

   * Addition (`+`)
   * Subtraction (`-`)
   * Multiplication (`*`)
   * Division (`/`)
4. The program performs the selected operation and displays the result.

## Concepts Covered

* User Input and Output
* Variables and Data Types
* Arithmetic Operators
* Switch-Case Statements
* Basic Program Flow
* Conditional Logic

## Purpose

The purpose of this project is to strengthen the fundamentals of C++ programming by implementing a simple calculator capable of performing basic mathematical operations. It helps beginners understand essential programming concepts while building a practical console application.
